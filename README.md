# Date Calculator

***

### Introduction

This is a simple C# project utilisting DateTime. I created this to practice C# and learn how to manipulate date and time in C#. The user may add or subtract any amount of time from the current date

***

### Instructions for use:
*The instructions should be clear within the code however this is for extra guidance.*

1) Enter a singular measurement of time
2) Enter a quantity
3) To add this quantity, enter 1
4) To subtract this quantity, enter 2
